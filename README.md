# Flappy Bird Clone

A fully responsive, mobile-friendly recreation of the classic Flappy Bird game built with vanilla HTML5, CSS3, and JavaScript. No frameworks or libraries - pure front-end magic!

- **Mobile-First Design**: Touch controls optimized for smartphones and tablets

- **Physics Engine**: Realistic gravity, collision detection, and momentum
- **Score System**: High score persistence with localStorage
- **Sound Effects**: Satisfying audio feedback (tap to toggle)
- **Game States**: Start screen, playing, game over with restart
- **Progressive Difficulty**: Pipes speed up as score increases
